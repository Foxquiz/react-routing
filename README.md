# Routing в React приложениях

Приложение для демонстрации.<br>
Состоит из нескольких страниц: главная, список пользователей, информация о конретном пользователе, список плейлистов, информация о конкретном плейлисте.<br>
Также есть возможность поиска пользователя по имени, поиск плейлиста по жанру и/или наименованию.<br>
Со страницы пользователя можно перейти на страницу плейлиста, со страницы плейлиста можно перейти на страницу с отфильтрованными по жанру плейлистами.

В качестве сборщика используется [vite](https://vitejs.dev/).

### Запуск режима разработки

```sh
npm run dev
```
